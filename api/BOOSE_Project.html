Namespace BOOSE_Project

Classes

AppArray

AppArray class extends the AppArray command in BOOSE.
It provides parameters checking for the array command.
When there are actions to the array,
it processes the array by recognising whether it is to get or set the value of an array item.

AppCanvas

AppCanvas class implements the ICanvas interface.
It provides several methods and properties to manage different operation in the drawing canvas.

AppCircle

AppCircle class implements the CommandOneParameter abstract class.
Overrides the Execute() and CheckParameters() methods to draw a circle, and validates the parameters.

AppDrawTo

AppDrawTo class implements the CommandTwoParameters abstract class.
Overrides the Execute() and CheckParameters() methods to draw a line, and validates the parameters.

AppElse

AppElse class overwrites the Else command in BOOSE.
serves for if-else conditional statements.

AppEnd

AppEnd class extends the End command in BOOSE.
detects the corresponding compound command to end.
Supports If, While, For, and Method commands.

AppFor

AppFor class extends the For command in BOOSE.
Overrides the base class ConditionalCommand.

AppIf

AppIf class extends the If command in BOOSE.

AppInt

AppInt class extends the Evaluation command in BOOSE.
it handles integer variable declaration and checks for valid integer assignment.

AppMethod

AppMethod class extends the Method in BOOSE.
it uses the original methods from the base class.

AppMoveTo

AppMoveTo class implements the CommandTwoParameters abstract class.
Overrides the Execute() and CheckParameters() methods to move the drawing cursor, and validates the parameters.

AppParser

AppParser class extends the Parser class in BOOSE.
overrides the ParseCommand() method to parse user input.

AppPeek

AppPeek class extends the AppArray command in BOOSE.
It handles the PEEK command to assign a value to an array element.
Uses the base class methods to process parameters for PEEK operation.

AppPenColour

AppPenColour class implements the CommandThreeParameters abstract class.
Overrides the Execute() and CheckParameters() methods to change the pen color, and validates the parameters.

AppPoke

AppPoke class extends the AppArray command in BOOSE.
It handles the POKE command to assign a value to an array element.
Uses the base class methods to process parameters for POKE operation.

AppReal

AppReal class extends the Evaluation command in BOOSE.
it handles Real variable declaration and checks for valid Real assignment.

AppRect

AppRect class implements the CommandTwoParameters abstract class.
Overrides the Execute() and CheckParameters() methods to draw a rectangle, and validates the parameters.

AppTri

AppTri class implements the CommandTwoParameters abstract class.
Overrides the Execute() and CheckParameters() methods to draw a triangle, and validates the parameters.

AppWhile

AppWhile class extends the While command in BOOSE.

AppWrite

AppWrite class implements the CanvasCommand abstract class.
Overrides Execute() to evaluate its parameters, and draw a string on the canvas.
Overrides CheckParameters() to validates the parameters.

ThePicassoApp

thePicassoApp class is a drawing app that does several basic operations.
Such as: drawing shapes with mouse cursor or typed in commands, changing pen color and stroke width, and clearing the canvas.
Additionally, it supports syntax highlighting in the text command input field.
